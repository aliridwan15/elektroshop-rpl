## ğŸš€ Cara Instalasi

1.  **Clone Repositori**
    ```bash
    git clone [https://github.com/aliridwan15/elektroshop-rpl.git](https://github.com/aliridwan15/elektroshop-rpl.git)
    ```
2.  **Siapkan Database**
    * Buka phpMyAdmin.
    * Buat database baru bernama `elektroshop`.
    * Import file `.sql` yang ada di folder database.
3.  **Konfigurasi**
    * Sesuaikan file koneksi database (`koneksi.php` atau `config.php`) dengan user/password MySQL lokal Anda.
4.  **Jalankan**
    * Buka browser dan akses `http://localhost/elektroshop-rpl`.
